

## Favicon Plan

Create a simple SVG favicon with a bold "T" lettermark using the Tokeniz brand colors (dark background with blue accent).

### What will be done:
1. **Create an SVG favicon** (`public/favicon.svg`) — A rounded square with the dark brand background (`#0a0f1a`) and a bold "T" in the primary blue (`#3b82f6`)
2. **Update `index.html`** — Replace the current `.ico` favicon reference with the new SVG

### Technical details:
- SVG format for crisp rendering at all sizes
- Colors: background `#0a0f1a` (from `--background`), text `#3b82f6` (from `--primary`)
- Font: sans-serif bold "T"
- Rounded corners on the square
